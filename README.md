# AgriGig 🌾

**Transform Farming into Micro-Actions** — Track, verify, and reward sustainable farming practices.

🔗 **Live Demo:** https://agri-gig-framework.vercel.app/

## About

AgriGig turns field work into measurable, rewarded progress by creating micro-gigs for farming tasks — from soil preparation to harvest. Workers upload proof of completion with photos and location data, and get rewarded instantly for verified sustainable practices.

- 👨‍🌾 **Farmers** create micro-gigs for farming tasks
- 👷 **Laborers** complete tasks and verify with photo/location proof
- 💰 Instant rewards for verified work

Built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

## Getting Started

First, install dependencies and run the development server:

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
